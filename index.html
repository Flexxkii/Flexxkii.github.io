Portfolio Website

16 / 01 / 2019

Dit is de site waar je op dit moment op bent. Mijn goal met dit website is jou rondleiden door mijn werk en ook mij leert kennen.
Om mijn skills goed te laten zien heb ik veel moeite in gestoken om dit site responsive & mooi te maken met animaties. Met kleuren heb ik secties op het site verdeeld maar ik hou een uniform primaire kleur en dat is grijs.
Op het contact sectie heb ik elke actie een visual gegeven door kleuren te gebruiken zodat de gebruiker weet dat hij de juiste informatie aangeeft en juiste formaat gebruikt.
Naast de code heb ik de site zo gedesigned dat elke sectie leuk uizit. en informatiefvol is. Hiermee wil ik de gebruiker m'n skills tonen.
